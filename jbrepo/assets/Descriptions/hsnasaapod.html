HSNasaAPOD - More details


HSNasaAPOD is a tweak that shows that day's APOD in a HSWidget. It has a few preferences, which I'll talk about below.

  • API Key - Set the Nasa Open API key. The default key is limited to 50 requests an hour (which should be enough for most people).
  • Enable HD Downloads - Download HD versions of images if possible. Disabled by default.
  • More info on Today's APOD - Not really a preference, but shows more info on that days APOD.
  • Refresh View - Refreshes that day's APOD. This fails for around 3 hours each day as that days APOD isn't chosen yet.


BUGS

  • It doesn't refresh the APOD on it's own, so you have to do it yourself by going into the preferences and hitting refresh.

That's pretty much it! Feel free to open an issue on the github here
